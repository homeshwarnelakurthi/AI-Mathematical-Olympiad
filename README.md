## 📌 Project Overview

This repository documents participation in the **AI Mathematical Olympiad (AIMO)** Kaggle competition — one of the most challenging AI benchmarks, requiring models to solve **olympiad-level mathematical problems** that demand deep multi-step reasoning, symbolic manipulation, and proof construction.

---

## 🏆 Competition Context

> The AIMO challenge pushes the frontier of AI reasoning. Unlike standard ML problems, this competition tests whether AI models can match human mathematicians on complex, open-ended problems across algebra, number theory, combinatorics, and geometry.

- **Problem Type:** Open-ended multi-step mathematical reasoning
- **Evaluation:** Exact answer matching on held-out problem sets
- **Difficulty:** International Mathematical Olympiad (IMO) standard
- **Platform:** Kaggle

---

## 🎯 Approach

The competition explores techniques including:

- **Chain-of-Thought Prompting** — guiding LLMs through step-by-step mathematical reasoning
- **Self-consistency Decoding** — sampling multiple solutions and selecting the most frequent answer
- **Tool-augmented Reasoning** — using Python code execution (via sympy, scipy) to verify algebraic steps
- **Fine-tuned Mathematical LLMs** — leveraging models specialised for formal mathematics
